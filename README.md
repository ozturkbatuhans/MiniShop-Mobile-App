# MiniShop Mobile App

Exam assignment for Mobile Application 2.

## Tech stack
- Expo + React Native
- TypeScript
- Expo Router
- TanStack Query
- Redux Toolkit

## Features
- Bottom tabs navigation (Home / Cart / Profile)
- Product list screen
- Product detail screen (dynamic routing)

## How to run
```bash
cd MiniShop
npm install
npm run start
